# icicSim: gemtek testbed ICIC simulation
@ Kevin Cheng a.k.a kevin5566, NTUEE 2017

## Compile
* download `main.cpp` and `Def.h`
* `g++ main.cpp -o icicSim`

## Execute
* download `input.txt` 
* `./icicSim input.txt`
